    Why I chose Twitter Bootstrap?


    When I sat down to plan my website for the coding challenge, to be honest my biggest constraint was time. Bundled up between my course work and on campus job, I had to figure out an effective way to be able to fulfill this challenge.


    I gathered my thoughts and remembered having used Twitter Bootstrap for one of my projects. I was familiar with using it, it is convenient and easy to use, it has some really cool features [responsiveness - important feature] and it therefore was the best framework to execute my idea.
